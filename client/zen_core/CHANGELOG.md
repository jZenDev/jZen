# Changelog

## 0.1.0

- Initial release. `ZenResult`/`ZenError`, `ZenLogger` (with a platform-selected strategy),
  value objects (`ZenTimestamp`, `ZenLocale`, `EmailAddress`, `UserId`), guards (`ZenGuard`,
  `ZenTry`), `BaseResponse`, the supported locale set (`ZenLocales`), and the compile-time
  `ZEN_*` env/platform constants.
